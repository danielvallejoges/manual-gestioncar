Roles y permisos

GestionCar usa un sistema de roles que define qué acciones puede realizar cada usuario. Los roles se asignan al crear o editar un usuario en Administración → Usuarios.

RolNivel de acceso
Super AdministradorAcceso completo a todas las funciones y configuraciones del sistema
AdministradorAcceso completo excepto remuneraciones, EERR, plan de cuentas y gestión de empresa
Jefe de TallerOTs, presupuestos, clientes, vehículos, inventario, compras. No puede gestionar usuarios ni sucursales
VendedorClientes, OTs, presupuestos, facturación, calendario. Descuento máximo configurable por usuario
AgendadorCalendario, citas, clientes y vehículos
MecánicoOTs asignadas, registro de horas, incidencias, checklists, inventario (solo lectura)
Encargado de InventarioInventario completo, ítems, movimientos, recepción de compras, escaneo de productos
Encargado de ComprasGestión de compras, proveedores, ítems, inventario (solo lectura)
ℹ️
El campo Máximo descuento autorizado en la ficha del usuario solo es visible para el rol Vendedor. Permite limitar el porcentaje de descuento que ese usuario puede aplicar sin aprobación.
